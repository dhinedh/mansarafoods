## How to Use

1. **Homepage**: Browse featured products, offers, combos, and new arrivals
2. **Login**: Use one of the credentials above
3. **Shopping**:
   - Browse products by category
   - View product details
   - Add items to cart (requires login)
   - Proceed to checkout
4. **Admin Panel** (admin login required):
   - View all products
   - Manage orders
   - View customers
   - Access dashboard statistics

## Features Available

### Customer Features
- Browse 10 pre-loaded products (Porridge Mixes & Oil/Ghee)
- View special offers (discounted products)
- Browse combos (bundled products with savings)
- Add products to cart
- Complete checkout process
- View order history (stored in browser localStorage)

### Admin Features
- Dashboard with statistics
- Product listing
- Order management
- Customer listing
- All data stored in browser localStorage

## Data Storage

All data is stored in your browser's **localStorage**:
- User sessions
- Shopping cart
- Orders
- Profile updates

**Note**: Clearing your browser data will reset everything to defaults.

## Quick Start

1. Open the application
2. Login with customer credentials
3. Add products to cart
4. Complete checkout
5. View your orders in Account page

Or login as admin to access the admin panel at `/admin`
